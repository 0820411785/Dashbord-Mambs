Publication du dashboard incidents - 12 juin 2026

Option recommandee pour Helios Towers : SharePoint / OneDrive / Teams

1. Ouvrir OneDrive ou SharePoint dans le navigateur.
2. Creer un dossier, par exemple : Dashboard Incidents.
3. Uploader tous les fichiers de ce dossier :
   - index.html
   - Dashboard_Incidents_12_Juin_2026_Incidents.csv
   - Dashboard_Incidents_12_Juin_2026_BySite.csv
   - Dashboard_Incidents_12_Juin_2026_ByHour.csv
   - Dashboard_Incidents_12_Juin_2026_Fuel.csv
4. Partager le fichier index.html ou le dossier avec les personnes autorisees.
5. Choisir un partage securise :
   - Personnes dans Helios Towers
   - ou personnes specifiques

Important :
- Eviter un lien public si le fichier contient des donnees internes.
- Le lien local http://10.243.253.38:8080/ ne fonctionne que sur le meme reseau.
- Pour un acces depuis la maison, il faut un lien cloud ou intranet accessible a distance.

Alternative plus professionnelle :
- Power BI Service pour un vrai dashboard online avec filtres et mises a jour.
- SharePoint page si l'entreprise veut une page interne simple.
